1. Логгирование ошибок происходит в файл с названием Logs.txt
Распологается файл рядом файлом решения(.sln)

2. Структура файла, данные из которого считываются данные, 
используемые для задания размеров зубчатых массивов:
Пример: Example.txt(название)
Пример структуры данных, которые должны содержаться в файле:

1
2
3
4
5

3. Программа в основном запрашивает абсолютные пути к файлам, вместе с 
их названием(расширение обязательно должно быть .txt)

Пример: 
C:\Users\username\source\repos\ProjectName\File.txt
